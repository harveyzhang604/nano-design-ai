## 生成策略

### 方案 1：使用 Nano Banana Pro (baoyu-image-gen)
- 优点：高质量，可控性强
- 缺点：需要逐个生成，耗时较长
- 适用：需要精确效果的功能

### 方案 2：使用现有 API 生成
- 优点：真实效果展示
- 缺点：需要测试图片
- 适用：可以自我生成的功能

### 方案 3：使用 Unsplash 真实照片
- 优点：快速，真实
- 缺点：可能不够精准
- 适用：通用场景
